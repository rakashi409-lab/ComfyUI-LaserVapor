01

FEATURES

/ 功能
F.01

文生图 / 图生图

上传本地参考图，或直接读取当前 Krita 画布作为输入，自动上传并注入到工作流的 LoadImage 节点。

F.02

提示词 & 尺寸

正向提示词写入 CR Text 节点，输出宽高一键覆盖尺寸节点，节点 ID 可按你的工作流自由配置。

F.03

结果直接入画布

生成图自动作为新图层贴回画布，或在新文档打开。多张结果以缩略图网格呈现，逐张操作。

F.04

原生 · 零依赖

纯 Python + Krita 自带 PyQt5 编写，无需 pip 安装任何库。折叠式面板，实时进度与日志。

02

INSTALL

/ 安装
01

下载 ZIP

点上方按钮下载 .zip，不要解压。

02

导入插件

Krita 菜单 → 工具 → 脚本 → 从文件导入 Python 插件 → 选中 zip。

03

启用

设置 → 配置 Krita → Python 插件管理器 → 勾选 农用激光除杂车 / Laser Vapor。

04

重启 Krita

启用插件后必须重启才能生效。

05

打开面板

设置 → 工具面板(Dockers) → 勾选插件，面板出现在右侧。

使用：面板填入 LibLib AccessKey / SecretKey 与 templateUuid / workflowUuid，选择输入方式，写提示词，点「⚡ 生成」。 国内若图床上传失败，在「高级 → 代理」填 http://127.0.0.1:7897。
