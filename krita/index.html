ComfyUI-LaserVapor

Krita 插件 · v1.0

在 Krita 内直接调用 LibLib ComfyUI 云端工作流生成图像

⬇ 下载插件 comfyui-laservapor-krita.zip 适用于 Krita 5.x（Windows / macOS / Linux）

✨ 功能

文生图 / 图生图支持上传本地参考图或使用当前画布
提示词 & 尺寸正向提示词、输出宽高一键覆盖工作流节点
结果直接入画布生成图自动作为新图层，或在新文档打开
折叠式面板配置、输入、结果分区收起，界面清爽

📦 安装步骤

  1. 1下载上方 .zip（不要解压）
  2. 2打开 Krita → 菜单 工具 → 脚本 → 从文件导入 Python 插件，选中刚下载的 zip
  3. 3设置 → 配置 Krita → Python 插件管理器，勾选 ComfyUI-LaserVapor
  4. 4重启 Krita
  5. 5设置 → 工具面板(Dockers) → 勾选 ComfyUI-LaserVapor，面板出现在右侧
手动安装：把 zip 内的 comfyui_laservapor 文件夹和 comfyui_laservapor.desktop 放到 %APPDATA%\krita\pykrita\（Windows），再从第 3 步开始。

🚀 使用

  • 面板①填入你的 LibLib AccessKey / SecretKey 和 templateUuid / workflowUuid
  • ②选择图片输入方式：文生图 / 上传本地图 / 用当前画布
  • ③填提示词、输出尺寸（节点 ID 按你的工作流填）
  • 点 ⚡ 生成，进度实时显示，完成后结果出现在「结果」区，可一键导入画布
国内若图片上传失败（图床被墙），在面板「④高级 → 代理」填 http://127.0.0.1:7897。 LibLib API 本身国内可直连。
